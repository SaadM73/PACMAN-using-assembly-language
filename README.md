# Pac-Man Assembly Game

Welcome to Pac-Man Assembly Game, a classic arcade game implemented in x86 assembly language. This game features three levels with different layouts, portals, and challenges inspired by the original Pac-Man.

## Features

- Three challenging levels with increasing difficulty.
- Classic Pac-Man gameplay mechanics.
- Unique maze layouts and obstacles.
- Portals for teleportation across the maze.
- Score tracking and high score display.
- Colorful graphics and retro arcade feel.

## Getting Started

### Prerequisites

- DOS emulator or environment for running assembly programs.

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/pacman-assembly-game.git
   ```

2. Set up a DOS environment or emulator like DOSBox.

3. Compile and run the game using the appropriate commands for your assembly environment.

## Gameplay

- Use the arrow keys to navigate Pac-Man through the maze.
- Collect dots and avoid ghosts to advance through the levels.
- Use portals strategically to move across the maze quickly.
- Reach high scores by clearing levels efficiently and collecting bonuses.

## Controls

- Arrow Keys: Move Pac-Man (Up, Down, Left, Right)
- Spacebar: Pause/Resume Game
